## Visão Geral

Página com um temporizador que executa uma contagem regressiva iniciando com 5 minutos a partir do clique de play.

## Tecnologias Utilizadas

- HTML
- CSS
- JS

## Funcionalidades

- Iniciar contagem regressiva
- Pausar contagem
- Retomar contagem do início
